Who am I

I'm an aspiring software engineer based in Massachusetts, fluent in both English and Hebrew. Born in Israel, I developed a passion for computer science, with interests in game development, creating innovative Python programs, and web development.

Python

I’ve been coding in Python since 2020 and have worked on a variety of projects . I’ve built applications that use computer vision, created tools for converting data between different formats, and developed several other intresting programs. Alongside my projects, I lead a non-profit organization dedicated to teaching python to middle school students.

Web development

I began web development in 2021 and have built several websites, including one for my non-profit and this website. I have experience with frameworks like React and Vue for creating dynamic, responsive sites. I also handle all the designs for my websites.

Unity

I began learning Unity in 2022 and have since developed and published a full game. I also participated in the GMTK Game Jam in 2023 and 2024, collaborating with friends to sharpen my development skills while enjoying the creative process.

Projects

YouStorage

a unique tool that converts text files to video files and vice versa, with additional functionality to download YouTube videos. It encodes text into binary (using Huffman coding) and represents it as pixel data compiled into video, or extracts text from video pixel data.
